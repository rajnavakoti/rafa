# Base checks

## What is this?

Base checks are validating if entities in the catalog are adhering to common agreed upon hygiene validation rules. Intention with the checks is to provide short feedback loops in order to build and maintain a consistent catalog cross the organisation.

Below sections covers the current implemented base checks, and how to fix validation errors for them respectively.

## Group owner check

Verifies that entity has owner set to a valid group

[Read more](./owner.md)

## Lifecycle check

Verifies that entity has lifecycle set to a valid value

[Read more](./lifecycle.md)

## SMC (System Master Catalog) system check

Verifies that entity has system set to a valid system in System Master Catalog (SMC)
